1.- Clonar el proyecto en la siguiente ruta C:\xampp\htdocs

2.- Abrir XAMPP y habilitar los servicios de "Apache" y "MySQL"

3.- Abrir el proyecto en Visual Studio Code, en la terminal ejecutar el comando: php spark serve

PARA LA VISTA PRINCIPAL

4.- En el navegador de su preferencia ejecutar la url: http://localhost:8080/

PARA LA VISTA DE ADMINISTRADOR

5.- En el navegador de su preferencia ejecutar la url: http://localhost:8080/admin/

PARA DETENER EL SERVIDOR

6.- Entrar a Visual Studio Code y presionar Ctrl+c
7.- Deshabilitar los servicios de "Apache" y "MySQL"
